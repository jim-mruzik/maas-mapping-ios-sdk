buildingWithIdentifier:usingCache:completion:

Instantiates a new building structure using the MaaS Portal identifier. The completion handler is called when the building structure is fully loaded, including any network access.

+ (void)buildingWithIdentifier:(NSInteger)identifier usingCache:(BOOL)caching completion:(void ( ^ ) ( PWBuilding *building , NSError *error ))completion

Parameters

identifier

The identifier used in MaaS Portal for the building.

caching

YES/NO. Determines if the SDK will use prior cache information to instantiate the building structure. Default YES.

completion

Completion handler that is called once building load is complete.

discussion

Please change to use [PWBuilding buildingWithIdentifier:completion:] instead.
